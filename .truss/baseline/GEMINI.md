# Gemini CLI — Truss Workspace

This project uses the open `agents.md` standard. Read `AGENTS.md` — that file is the complete boot instruction for this workspace.

> **Important Setup:** This workspace relies on local scripts (`.truss/bin/truss.mjs`) to manage its state and health. You must ensure the Gemini CLI / Agent has permission to execute terminal commands automatically in this directory, otherwise the system will fail.
